Project Title:

 Digital Evidence  Management & Crime Record Tracking System

Digital Evidence Management and Crime Record Tracking System is an application developed to manage criminal records and digital evidence in a secure and organized way. It allows users to store case information, track investigations, upload evidence files, and access records quickly for better case management and reporting.

Project Overview:

This project is a Python-based console application designed to help police departments manage crime records and digital evidence efficiently. The system allows officers or administrators to store, update, search, and track criminal case information in an organized way.

The application focuses on managing investigation details such as case IDs, suspect information, evidence records, assigned officers, and case status. It also provides search and filtering features to quickly find cases and generate investigation reports.

The system stores data permanently using JSON or CSV files so records remain available even after the program is closed.

Main Features:

1)Add and manage crime cases

2)Track digital evidence (images, videos, documents, etc.)

3)Update investigation status
Search and filter crime records

4)Generate investigation reports

5)Save and load records automatically

6)Handle invalid or duplicate data safely

6)Technologies Used
Python

7)File Handling (JSON/CSV)
Object-Oriented Programming (OOP)

Modules Included:

1)main.py – Main program execution

2)case_manager.py – Crime case management

3)evidence_manager.py – Evidence tracking

4)report_generator.py – Report creation

5)utils.py – Helper functions


Project Objective:

The main objective of this project is to improve crime record organization and digital evidence tracking through a simple software solution while giving practical experience in Python programming, file handling, and database-like record management.
